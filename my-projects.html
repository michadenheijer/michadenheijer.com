CissTM

I'am building a satellite as a schoolproject. We're participating in the Dutch CanSat competition, and we're hoping that to be able to track objects on the ground using signal strength and the speed of light. I'm the main programmer and I've decided to use a Raspberry Pi zero as it's main computer. The software is programmed in Python (3.6) and the groundstation can display the data live using a NodeJS program connected to a Arduino Uno that receives the data.

Underground

I've made a website that allow you to read and update data displayed on your website without the need of reloading the page. It's quite fast, and it allows the HTML to be cashed. It uses a MySQL database and NodeJS as the server. The data is send by a module named Socket.IO and it allows for quite big browser compatebility while still using WebSockets. I havn't designed it myself, but I have used the Evie theme from Undraw.
